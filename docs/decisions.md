# Journal des décisions

Format :

```
## AAAA-MM-JJ — Titre

**Contexte** : ...

**Décision** : ...

**Conséquences** : ...
```

Ajouté par Claude sur validation humaine, ou à la main. Ce journal n'est
pas chargé à chaque session — consulté à la demande (voir la référence
dans [CLAUDE.md](../CLAUDE.md)).
