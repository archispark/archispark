# ArchiSpark — MCP Server

Model Context Protocol server for ArchiSpark, an open-source ArchiMate 3.1 modeling tool.

Exposes 38 tools, 2 prompts and 2 resources over the MCP Streamable HTTP transport, enabling AI agents to create and manage ArchiMate models.
